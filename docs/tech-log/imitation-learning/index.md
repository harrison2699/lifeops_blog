# Imitation Learning

**Imitation learning (IL)** covers behavior cloning, DAGGER-style aggregation, inverse RL, and demonstration-conditioned policies—anything where **expert data** (teleop, kinesthetic teaching, scripted experts) shapes the policy without a hand-designed reward.

## Scope for this section

- Dataset design, noise, and **causal confusion** in BC  
- On-policy correction and **interactive** imitation  
- Bridges to RL (offline RL from demos, hybrid fine-tuning)
