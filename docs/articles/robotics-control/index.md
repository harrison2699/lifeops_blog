# Robotics Control

Articles in this category focus on **making robots move well**: modeling assumptions, controller structures, contact and impedance, safety limits, and what breaks first on real hardware.

Expect write-ups on tuning discipline, predictable failure modes, and comparisons that respect implementation cost—not only asymptotic theory.

When I publish here, the goal is a note I would still trust after swapping motor drivers or revisiting the same stack a year later.
